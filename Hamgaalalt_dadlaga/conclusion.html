Дүгнэлт

Үйлдвэрлэлийн дадлагын үр дүн, сурсан зүйлс

Ололт амжилт

Бодит төсөл дээр ажиллаж, практик туршлага олсон. Багийн болж ажиллах, код review хийх зэрэг мэргэжлийн ур чадвар эзэмшсэн.
  • Бодит төсөл дээр ажиллах туршлага
  • Баг болж хамтарч ажиллах
  • Production код бичих

Сурсан зүйлс

Эмнэлгийн системийн хөгжүүлэлт дээр ажиллаж, архитектур, компонент дизайн, REST API холболтын арга техник сурсан.
  • Next.js, TypeScript
  • REST API холболт
  • System architecture
Дадлагын үр дүн
Багийн ажиллагаанд идэвхтэй оролцож, төслийн менежмент, практик туршлага, системийн найдвартай байдлыг хангах арга барилуудыг сурсан.
Цаашидаа хийх зүйлс
Олж авсан мэдлэгээ ашиглан илүү нарийн төслүүд хөгжүүлэх, open source төслүүдэд оролцох, веб хөгжүүлэлтийн чиглэлээр мэргэших.

Ийнхүү "Синдүра Мед" систем дээр хийсэн дадлага нь миний мэргэжлийн ур чадварыг нэмэгдүүлж, цаашид веб хөгжүүлэлтийн чиглэлээр ажиллах хүсэл тэмүүллийг улам бататгалаа.
